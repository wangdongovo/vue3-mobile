# vue3-mobile

这个模板可以在 Vite 中开始使用 Vue 3 进行开发

## 技术栈

```ts
Vue3
vue-router
pinia
Typescript
Vite
Less
```

代码提交前做prettier处理统一编码风格

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
